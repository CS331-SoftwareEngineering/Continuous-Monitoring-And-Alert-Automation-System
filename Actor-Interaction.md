# Actors and Their Interactions

## Registered Users

Represents registered end users who consume alerts and system information.

### Interactions:
- User Login
- User Profile Management
- View Dashboard
- View Alerts
- Acknowledge Alerts
- Change Threshhold
- Change Notification Settings
- Data Filtering and searching

## External Systems

External systems or APIs act as supporting actors that integrate with the Monitoring & Alert Automation System.

### Interactions 
- Login
- DashBoard View
- Historical Data View
- Data Filtering and Searching
- Alerts Management
- Alert Acknowledgement
- Threshold Change
- Notification Options
